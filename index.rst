Welcome to EVPV-Simulator's Documentation
=========================================

The **EVPV-Simulator** is an open-source python tool to support privately owned vehicle electrification planning studies in a given region of interest.

Here, you can find the extended documentation of the model, with user guides, examples, and a full API reference for advanced use. Please make sure to also check the [GitHub organization](https://github.com/evpv-simulator) containing all the repositories of the project. If you prefer, a concise version of the documentation is also available as a README file on the main model repository.

.. note::
   This project is under active development. Expect some small mistakes and empty entries. Do not hesitate to report any issue!
